## 現行UIの主要機能と移行後チェック項目
以下は現行UIで提供している主要機能の一覧と、移行後に確認すべきチェック項目です。

### 主要機能の一覧
- タブ切替（翻訳/ログ/設定などの主要ビュー間移動）
- ファイル読み込み（翻訳対象データのロード）
- ファイル保存（編集内容の保存/エクスポート）
- 翻訳編集（原文/訳文/メタ情報の編集）
- ログ確認（処理ログ・警告・エラーの閲覧）
- 辞書ビルド（辞書の生成/更新）

### 移行後チェック項目（UI）
- [ ] タブ切替が正常に機能し、遷移先の表示が崩れない
- [ ] ファイル読み込み後に内容が正しく表示される
- [ ] ファイル保存が成功し、保存内容が再読込で再現される
- [ ] 翻訳編集の入力が即時反映され、保存できる
- [ ] ログ表示が最新状態に更新され、警告/エラーが識別できる
- [ ] 辞書ビルドが完了し、結果がUIに反映される

### 不具合報告時の再現手順（簡潔）
1. 使用環境（OS/バージョン、アプリのビルド種別、実行方法）を記載。
2. 操作手順を時系列で列挙（例: タブA→ファイル読み込み→編集→保存）。
3. 期待結果と実際結果を短く比較。
4. 可能なら対象ファイル/ログ/スクリーンショットを添付。
